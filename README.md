\# 淘宝用户消费行为分析（2021-2023）

\## 项目简介

本项目基于淘宝2021年1月至2023年3月的99,457条真实交易数据，从用户画像、品类分析、RFM用户分层、时间趋势四个维度进行深度分析，并提出可落地的运营建议。

\## 核心结论

\- \*\*用户画像\*\*：女性用户贡献1.15亿（53%），35-44岁人均消费最高（2,519元）

\- \*\*品类表现\*\*：服装为绝对主力（1.02亿），科技品类客单价最高（2,904元）

\- \*\*用户价值\*\*：高价值用户仅占8.1%，贡献19.7%销售额

\- \*\*时间规律\*\*：周一为周销售高峰，10月/12月为年度高点
